# AI Code Guide 项目报告

## 项目概览

**项目名称**: AI Code Guide
**作者**: Vilson Vieira (@aut0mata) 和 Eric S. Raymond (@esrtweet)
**许可证**: MIT
**项目类型**: 技术文档/教育指南

## 执行摘要

AI Code Guide 是一个全面的开源指南，旨在帮助开发者和非技术人员理解和使用 AI 辅助编程工具。该项目涵盖了从基础概念到高级实践的完整知识体系，为不同层次的用户提供了清晰的学习路径。

## 项目目标

1. **教育目的**: 为开发者提供关于 AI 编程工具的全面指南
2. **实用导向**: 提供可操作的最佳实践和工作流程
3. **社区驱动**: 通过开源协作持续更新和改进内容
4. **降低门槛**: 帮助非技术人员通过"vibe coding"进入编程领域

## 内容结构分析

### 1. 核心概念部分 (20%)
- AI 编程的定义和分类
- Vibe coding 概念解释
- AI 作为副驾驶 vs. AI 作为主驾驶的对比

### 2. 实践指南部分 (40%)
- **工具选择指南**:
  - Web 工具: Bolt, Replit, v0, Lovable
  - 桌面编辑器: Cursor, Windsurf, VSCode + Copilot
  - CLI 工具: aider, Claude Code, OpenAI Codex
  - 开源方案: OpenHands

- **Prompt 工程**:
  - PRD (产品需求文档) 创建方法
  - 任务分解策略
  - Markdown/XML 标签使用

- **项目设置**:
  - Frontend 技术栈选择
  - Backend 架构设计
  - 游戏开发特殊考虑

### 3. 进阶主题部分 (25%)
- **错误处理**: 调试策略和工具集成
- **协议与标准**: MCP、SLOP、A2A 协议解析
- **代码质量**: TDD、属性测试、安全实践
- **架构设计**: 分层设计、关注点分离

### 4. 资源汇总部分 (15%)
- 70+ 外部资源链接
- 工具分类列表 (编辑器、CLI、Web应用)
- 行业专家推荐关注列表
- 社区贡献指南
